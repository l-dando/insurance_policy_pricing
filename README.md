## About The Project

<!--[![Product Name Screen Shot][product-screenshot]](https://example.com)-->

mathematical aspect, or it delves in much deeper than necessary. The idea of this project is to gradually introduce how 
one would go about building their own pricing models and even neural networks "by hand" so to speak.

## Getting Started

This project is entirely LaTeX based, and so importing it into any relevant compiler will produce a `.pdf` file of the 
work done. Note that any `.pdf` found within the project files themselves may not be up to date with the `.tex` file
and so compiling yourself may reward a better document.

### Prerequisites

A strong understanding of higher education mathematics (differentiation, advanced algebra etc.) is required, but there a 
is a vast amount of information and help online to both teach you or refresh your memory.

As well as this, basic knowledge on LaTeX is required if you wish to add or make changes to the document itself.

### Installation

1. Get access to some kind of `.tex` compiler, such as [Overleaf](https://www.overleaf.com).
2. Clone the repo, or download & import the project to a relevant editor.
3. Either compile the `.pdf` or make changes etc.
